# Creating an app

python manage.py create_cleanapp app_name -p ib_miniprojects_backend

# Building an app

python manage.py build -a app_name

# Create a snapshot test file

python manage.py create_snapshottest -a app_name -o create_new_post -n 3

# Executing Tests

  • You need not use python manage.py test command.
  • Use only pytest and pytest --snapshot-update commands to run tests and update snapshots respectively.
  • According to the new DSU, You should write views related snapshot tests in app_name/tests/views package
  • Instead of default_test_case you should be calling make_api_call
  • Instead of setupuser you can now use api_user a pytest fixture
