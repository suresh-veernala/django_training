Privacy Mode

To ensure that our codebase references and prompts are not saved in provider servers follow below steps:

  1. Go to File → Preferences → Cursor Settings → General
  2. Scroll to the bottom
  3. Toggle the Privacy Mode setting to Enabled

Cursor Account Login
  1. Go to File → Preferences → Cursor Settings → General
  2. In the Account section, click Sign In
  3. Login using the provided email in your browser
  4. Once logged in, the email should appear under the Account section

Configure cursorignore

Configuring .cursorignore helps in ensuring large/irrelevant and secrets files are not indexed and sent to AI in context.

  1. Create a new file names .cursorignore in the project root directory.
  2. Copy the rules from .gitignore and paste it in the .cursorignore file.
  3. Add/Update the .cursorignore file to exclude unnecessary additional files such as csvs/large files or secrets which aren't defined in the .gitignore file.
  4. Ensure the .cursorignore file is also updated whenever .gitignore file is updated.
  5. Additional paths that can be ignored only in .cursorignore are:
    • *api_specs/*

Reference (opens new window)


Update gitignore
  1. Update .gitignore file with below.
    • .cursor/*
  2. Regularly update .gitignore file with any default files generated by cursor or extensions.

Codebase Indexing

Codebase Indexing helps in retrieving relevant context based on the requirement and being sent to AI.

  1. Navigate to File → Preferences → Cursor Settings → Features
  2. Scroll down and click Show Settings in the Codebase Indexing section
  3. Enable:
    • Index New folders by default
    • Git graph file relationships
  4. Delete existing index and reindex.

User Rules
  1. Navigate to File → Preferences → Cursor Settings → Rules → User Rules
  2. Copy and paste your predefined User Rules

Project Rules
  1. Navigate to File → Preferences → Cursor Settings → Rules → Project Rules
  2. Refer this docs (opens new window) to know how to create project rules.
  3. Create the project rules by copying the predefined rules

Optional Settings

Enable Large Context
  1. Navigate to File → Preferences → Cursor Settings → Features
  2. Scroll to Large Context and turn it on
  3. Note
    • Use this only if you need a larger context window based on the requirement. Know More (opens new window)
    • This will double the cost per request.

VS Code Import

If you're a previous VS Code user:

  1. Navigate to File → Preferences → Cursor Settings → General
  2. Click Import under the VS Code Import section to bring in extensions and settings
