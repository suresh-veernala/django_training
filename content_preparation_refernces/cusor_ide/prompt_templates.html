<CleanCodeInstructions>

This section outlines all the naming conventions, formatting rules, and code style guidelines you must follow. It ensures that your code output is clean, consistent, and aligned with the project’s standards.

<Naming>
1. Clarity and Intent:
   - Names should clearly answer why it exists, what it does, and how it's used.
   - Specify what is being measured and the unit of measurement.
   - Use appropriate names (e.g., plural for groups, suffixes like _dict, _list).
2. Differentiation:
   - For similar names, put variations at the beginning or end to differentiate easily.
   - Avoid number-series naming (a1, a2, ...) unless intended.
3. Readability:
   - Don't use noise words (e.g., data, info) for distinction.
   - Make variable names pronounceable.
   - Avoid cryptic abbreviations (e.g., genymdhms).
   - Don't use similar names with minor variations (e.g., kar instead of car).
</Naming>

<Readability>
1. Size and Complexity:
   - Limit functions to 20 lines (30 max for exceptions).
   - Keep indent levels to 3 or less (4 max for exceptions).
   - Limit blocks within if, else, while, and for statements to 4 lines max.
2. Simplicity:
   - Avoid flag arguments (boolean, enums, integers).
   - Keep conditional expressions simple; extract complex ones into functions.
   - Avoid negative conditionals (prefer positive phrasing).
3. Error Handling:
   - Limit try-catch blocks to 4 lines; extract longer bodies into separate functions.
4. Code Organization:
   - Group related functions together within a file.
   - Place the most important and frequently used functions at the top of the file.
   - Separate concerns within a file using clear, descriptive comments.
   - Use consistent spacing between functions and logical sections of code.
</Readability>

<ProjectCodeStyling>
1. Naming Conventions:
   - Class names for factories and DTOs must include "Factory" or "DTO" respectively.
   - Variable names of model or model factory instances should end with "_object".
   - Variable names of DTO or DTO factory instances need to end with "_dto".
=
2. Typing:
    - Include type information for all method signatures (input args & return) and instance variables.
   - Avoid typing for local variables.
   - Omit return type for methods that don't return anything. DO NOT add None as return type.
   - Typing is not needed for test methods.

3. Enum Organization:
   - Related Enums need to be grouped together in constants/enums.py.
   - Enums should have an Enum Suffix.

4. Utility Functions:
   - Code in the main utils module should be minimized.
   - Interacting with external services should be done in the utils module

 5. Misc
   - TODOs in the code are added intentionally.
   - Constants declared in constants should always be in capital letters.
   - Group functions in a class: public methods at the top, followed by private methods.
   - Declare variables in functions before their usage.
   - Maintain consistent variable names/conventions for the same value across different methods/places.
   - Pass only the required values from a method based on the context of its usage.

</ProjectCodeStyling>

</CleanCodeInstructions>

                continue

            if each.count("/") != 2:  # noqa: PLR2004
                invalid_service_enums.append(each)
                continue

            if service_provider not in \
                    AIServiceProviderEnum.get_list_of_values():
                invalid_service_enums.append(each)

        return invalid_service_enums

from typing import List, Any, Dict, Optional

from nkb_learning_resource.interactors.\
    create_or_update_tutorial_step_interactor\
    import CreateOrUpdateTutorialStepsInteractor
from nkb_learning_resource.interactors.storages.\
    tutorial_storage import TutorialStepDTO
from nkb_learning_resource.storages.tutorial_storage \
    import TutorialStorageImplementation
from nkb_learning_resource.utils.fetch_data_from_spread_sheet import \
    FetchDataFromSpreadSheet


class CreateOrUpdateTutorialStepsSheetUtil(FetchDataFromSpreadSheet):

    def create_or_update_tutorial_steps_from_sheet(
    self, spread_sheet_name: str,sub_sheet_name: str,
    spreadsheet_id: Optional[str]=None):

        tutorial_steps = self._format_spreadsheet_data(
            spread_sheet_name, sub_sheet_name, spreadsheet_id)
        storage_implementation = TutorialStorageImplementation()
        interactor = CreateOrUpdateTutorialStepsInteractor(
            storage_implementation)

        interactor.create_or_update_tutorial_steps(tutorial_steps)

    def _format_spreadsheet_data(
            self, spread_sheet_name: str, sub_sheet_name: str,
            spreadsheet_id: Optional[str]=None) -> List[Dict[str, Any]]:

        spread_sheet_data = self.fetch_data_from_spread_sheet(
            spread_sheet_name, sub_sheet_name, spreadsheet_id)
        tutorial_step_dicts = []
        row = 1
        while row < len(spread_sheet_data):
            data = spread_sheet_data[row]

            tutorial_step_dicts.append(
                {
                    "id": data[0].strip(),
                    "tutorial_id": data[1].strip() if data[1].strip() else
                    None,
                    "step_entity_id": data[2].strip() if data[2].strip() else
                    None,
                    "step_entity_type": data[3].strip()if data[3].strip() else
                    None,
                    "order": self._convert_string_to_integer(data[4].strip()),
                    "content": data[5].strip() if data[5].strip() else None,
                    "content_format": data[6].strip() if data[6].strip()
                    else None,
                },
            )
            row += 1

        return tutorial_step_dicts

    @staticmethod
    def _convert_string_to_integer(string: str):
        return int(string)

```
from typing import List, Dict, Any

from nkb_learning_resource.constants.constants import ContentType, \
    TutorialStepEntityType
from nkb_learning_resource.exceptions.custom_exceptions import \
    InvalidDataException
from nkb_learning_resource.interactors.storages. \
    tutorial_storage import TutorialStorage, TutorialStepDTO


class CreateOrUpdateTutorialStepsInteractor:
    def __init__(self, storage: TutorialStorage):
        self.storage = storage

    def create_or_update_tutorial_steps(
            self, tutorial_step_dicts: List[Dict]):

        self._validate_data(tutorial_step_dicts)

        tutorial_step_ids = [
            step_dict['tutorial_step_id']
            for step_dict in tutorial_step_dicts
        ]

        existing_tutorial_step_dtos = self.storage.get_tutorial_step_details(
            tutorial_step_ids)
        existing_tutorial_step_ids = [
            dto.tutorial_step_id for dto in existing_tutorial_step_dtos]

        tutorial_step_dtos_to_create = []
        tutorial_step_dtos_to_update = []
        for each in tutorial_step_dicts:
            if each['tutorial_step_id'] in existing_tutorial_step_ids:
                tutorial_step_dtos_to_update.append(
                    self._convert_to_tutorial_step_dto(each)
                )
            else:
                tutorial_step_dtos_to_create.append(
                    self._convert_to_tutorial_step_dto(each)
                )

        if tutorial_step_dtos_to_create:
            self.storage.create_tutorial_steps(tutorial_step_dtos_to_create)
        if tutorial_step_dtos_to_update:
            self.storage.update_tutorial_steps(tutorial_step_dtos_to_update)

    def _convert_to_tutorial_step_dto(
            self, tutorial_step: Dict[str, Any]) -> TutorialStepDTO:
        return TutorialStepDTO(
            tutorial_step_id=tutorial_step['tutorial_step_id'],
            tutorial_id=tutorial_step['tutorial_id'],
            step_entity_id=tutorial_step['step_entity_id'],
            step_entity_type=tutorial_step['step_entity_type'],
            order=tutorial_step['order'],
            content=tutorial_step['content'],
            content_format=tutorial_step['content_format']
        )

    def _validate_data(self, tutorial_step_dicts: List[Dict[str, Any]]):
        content_format = []
        entity_type = []
        question_set_ids = []
        question_ids = []
        tutorial_ids = []
        algo_visualization_ids = []

        for step_dict in tutorial_step_dicts:
            if step_dict['content_format']:
                content_format.append(step_dict['content_format'])

            if step_dict['step_entity_type']:
                entity_type.append(step_dict['step_entity_type'])

            if step_dict.get('step_entity_type') == TutorialStepEntityType.default_questions.value:
                question_set_ids.append(step_dict['step_entity_id'])

            if step_dict.get('step_entity_type') == TutorialStepEntityType.coding.value:
                question_ids.append(step_dict['step_entity_id'])

            if step_dict.get('step_entity_type') == TutorialStepEntityType.\
                    algo_visualization.value:
                algo_visualization_ids.append(step_dict['step_entity_id'])

            tutorial_ids.append(step_dict['tutorial_id'])

        self._validate_uuid_formats(tutorial_ids + question_ids + question_set_ids + algo_visualization_ids)

        invalid_content_formats = self._get_invalid_content_format(content_format)
        invalid_entity_type = self._get_invalid_entity_type(entity_type)

        invalid_tutorial_ids = self._get_invalid_tutorial_ids(tutorial_ids)

        if question_ids:
            self._validate_question_ids(question_ids)
        if question_set_ids:
            invalid_question_set_ids = self._get_invalid_question_set_ids(question_set_ids)
        if algo_visualization_ids:
            invalid_algo_visualization_ids = self._get_invalid_algo_visualization_ids(algo_visualization_ids)

        invalid_data = {}
        if invalid_content_formats:
            invalid_data["invalid_content_formats"] = invalid_content_formats
        if invalid_entity_type:
            invalid_data["invalid_entity_type"] = invalid_entity_type
        if invalid_question_set_ids:
            invalid_data["invalid_question_set_id"] = invalid_question_set_ids
        if invalid_tutorial_ids:
            invalid_data["invalid_tutorial_ids"] = invalid_tutorial_ids
        if invalid_algo_visualization_ids:
            invalid_data["invalid_algo_visualization_ids"] = \
                invalid_algo_visualization_ids

        if invalid_data:
            raise InvalidDataException(invalid_data)

    @staticmethod
    def _validate_uuid_formats(uuids: List[Any]):

        invalid_uuids = []
        for uuid in invalid_uuids:
            try:
                uuid.UUID(uuid, version=4)
            except Exception:  # noqa: PERF203
                invalid_uuids.append(uuid)

        if invalid_uuids:
            raise InvalidDataException(invalid_uuids)

    def _get_invalid_content_format(self, content_formats: List[str]) -> List[str]:
        return list(set(content_formats) - set(ContentType.get_list_of_values()))

    def _get_invalid_entity_type(self, entity_types: List[str]) -> List[str]:
        return list(set(entity_types) - set(TutorialStepEntityType.get_list_of_values()))

    def _get_invalid_tutorial_ids(self, tutorial_ids: List[str]) -> List[str]:
        existing_tutorial_ids = self.storage.get_existing_tutorial_ids(tutorial_ids)
        return list(set(tutorial_ids) - set(existing_tutorial_ids))

    def _get_invalid_question_set_ids(self, question_set_ids: List[str]) -> List[str]:
        from nkb_learning_resource.adapters.service_adapter import get_service_adapter

        adapter = get_service_adapter()
        existing_question_set_ids = adapter.question_service.get_question_set_ids(question_set_ids)

        return list(set(question_set_ids) - set(existing_question_set_ids))

    def _get_invalid_question_ids(self, question_ids: List[str]) -> List[str]:
        from nkb_learning_resource.adapters.service_adapter import get_service_adapter

        adapter = get_service_adapter()
        existing_question_ids = adapter.question_service.get_question_ids(question_ids)

        return list(set(question_ids) - set(existing_question_ids))

    def _get_invalid_algo_visualization_ids(self, algo_visualization_ids: List[str]) -> List[str]:
        from nkb_learning_resource.adapters.service_adapter import get_service_adapter

        adapter = get_service_adapter()
        existing_algo_visualization_ids = adapter.algo_visualization_service.get_algo_visualization_ids(algo_visualization_ids)

        return list(set(algo_visualization_ids) - set(existing_algo_visualization_ids))

```
