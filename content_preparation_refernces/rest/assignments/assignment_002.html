# Serialization

In this assignment you will be writing APIs which can handle JSON data.

# Submission Guidelines

  • Create a django project rest_assignment_002 in /home/ec2-user/environment/rest/rest_submissions
  • Create an app fb_post in the project.
  • Write all your views in view.py file of fb_post app.
  • utils package or module in your fb_post should expose the below functions
    • create_post
    • create_comment
    • reply_to_comment
    • react_to_post
    • react_to_comment
  • Complete Task1 and Task2 first.

# Task 1

Write an create_post API for the following details.

# Request

  • We want the API to be served on POST method
  • URL to be served 'http://localhost:8000/fb_post/post/'
  • Request Body
{
    "user_id": "ID of User",
    "content": "Content for the Post"        
}

# API Logic

  • The api should create a post with the given data (user_id, content) in the Database.

# Response

  • The api should return an HTTP Response with the following data in the Response body and Status codes based on the response.
{
    "post_id": "ID of the Post"
}
  • Status Codes:
    • If post is created - 201
    • Invalid user - raise 404
    • Invalid post content - raise 400
