        # Modelling Database for Real-World Scenarios

        # E-commerce Use-case

        A typical e-commerce web application consists of products and customers.

        A Product has the following information:

        • Name
        • Price
        • Description
        • Average Rating
        • Brand which manufactured the product

        Each Brand has

        • name
        • country
        • brand score

        Each Customer is defined by

        • name
        • mobile number
        • email address

        A customer can place multiple orders

        An Order can contain more than one product

        A customer can buy any quantity (number) of the same product in a single order

        Other information which an Order should capture is

        • the total value of the order
        • estimated date of delivery for each of the product
        • purchase date

        Each customer’s PAN details are also stored. Each user can have only one PAN detail.

        CustomerPANDetails has:

        • PAN Number
        • Name as per PAN
        • Date of birth
        • A boolean to check if the PAN details are verified or not.

        # Family Tree Use-case

        We have the following data of people

        • Name
        • Age
        • Gender.

        We also need to capture the “Parent - Child” relationships among these people.

        # Social Network Use-case

        A Social Network has users.

        A User is defined by

        • name
        • username - unique across all users
        • email id - unique across all users
        • profile pic

        Note: Images can be stored as their corresponding URLs in the database

        A user can make friends. Friendship is symmetrical, i.e., If user1 is friends with user2, then user2 is friends with user1 too.

        Each User can create a Post. A post contains some text and can include images.

        Any user can comment on a Post. A comment is some text and can contain images.

        Users can also react to posts with one of the following reactions

        • HAPPY
        • ANGRY
        • SAD
        • LIKE
        • HAHA
