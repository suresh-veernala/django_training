          # Guidelines to ease your learning

          DETAILS

          • In case of sqlite3 connection error run below command in every new terminal from which you access sqlite3
            export LD_LIBRARY_PATH=/usr/local/lib

          • You should run all your python3 manage.py {command} commands in your project folder i.e folder which contains manage.py

          • Use python3 manage.py ... where ever python manage.py ... is refered in the resources

          • Always run your server on 8080 port. By default your tutorials have 8000

            (venv)$ python manage.py runsever 8080

          • How to view your django server. Click on the Preview option and then select Preview Running Application.

            Note

            This works only when you run have run python manage.py runsever 8080 command exactly.

          • During your learning when you see an error as below i.e DisallowedHost ...

            Solution:
            Add “*” in ALLOWEDHOSTS.
            Open settings.py (opens new window) file in your project folder. And update it as below.

          import os  
...

#Build paths inside the project like this: os.path.join(BASE_DIR, ...)  
BASE_DIR = os.path.dirname(  
    os.path.dirname(os.path.dirname(os.path.abspath(__file__)))
)  

ALLOWED_HOSTS = [  
    "*"  
]
....
